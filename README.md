# meguminBot

## A discord bot based on the character Megumin from Konosuba.

Has the following commands: 

### Fun commands:
  - explosion
  - hi
  - subreddit
  - twitter
  - watch
  - website 

### Info commands:
  - help
  - no
  - server

### Moderation commands:
  - ban
  - kick
  - prune
  - reload
  - restart
  - unban
